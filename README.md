# Система інтелектуального розпізнавання та структурування волевиявлення
«Капелюх» — це цифрова екосистема, яка допомагає людині перетворити потік думок, мрій та намірів у чітку структуру. Проєкт поєднує в собі простоту Telegram-бота, потужність штучного інтелекту Gemini та аналітичну глибину веб-інтерфейсу.
