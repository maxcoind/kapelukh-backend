# Project Description

A scalable FastAPI backend service with real-time WebSocket support and Telegram bot integration.

## Key Features

- **REST API**: Authentication, payments, and Telegram users management
- **WebSocket Server**: Plugin-based real-time event streaming with subscription and filtering capabilities
- **Telegram Bot**: Full integration using aiogram with webhook and polling support
- **Database**: SQLAlchemy with SQLite (aiosqlite) and SQLModel ORM
- **External Services**: Extensible service registry for third-party integrations
- **Testing**: pytest with async support
